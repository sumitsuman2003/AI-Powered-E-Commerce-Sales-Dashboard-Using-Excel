# 📊 AI-Powered E-Commerce Sales Dashboard Using Excel

## 📌 Project Overview
This project is an interactive E-Commerce Sales Dashboard created in Microsoft Excel.  
The dashboard helps analyze sales performance, category-wise sales, regional profit, and monthly sales trends using Pivot Tables and Pivot Charts.

---

## 🚀 Features
- Data Cleaning & Preprocessing
- Duplicate Data Removal
- Monthly Sales Trend Analysis
- Category Wise Sales Analysis
- Region Wise Profit Analysis
- KPI Cards
- Interactive Dashboard Design
- Business Insights Generation

---

## 📈 Key Insights
- November generated the highest sales
- Technology category performed best
- West region generated highest profit
- Sales increased sharply in Q4
- High discounts caused some loss-making orders
